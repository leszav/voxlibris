# Деплой

В этом разделе описаны процессы и практики деплоя приложения VoxLibris.

## Содержание

- [Docker](docker.md) - Контейнеризация приложения
- [Сборка](build.md) - Процесс сборки приложения
- [Настройка окружения](environment-setup.md) - Конфигурация окружений
- [Мониторинг](monitoring.md) - Отслеживание состояния приложения

## Обзор

Процесс деплоя приложения VoxLibris включает в себя контейнеризацию с помощью Docker, настройку окружений, сборку приложения и мониторинг состояния системы. Приложение может быть развернуто как в облачной инфраструктуре, так и на собственных серверах.
