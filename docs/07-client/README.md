В этом разделе описаны компоненты клиентской части приложения VoxLibris.

- [Настройка](setup.md) - Установка и настройка клиентской части
- [Компоненты](components.md) - Описание UI-компонентов
- [Страницы](pages.md) - Описание страниц приложения
- [Интеграция с API](api-integration.md) - Взаимодействие с серверным API

## Архитектура клиентской части

Клиентская часть приложения реализована с использованием React 19, TypeScript и Vite. Она обеспечивает пользовательский интерфейс для взаимодействия с функциями платформы VoxLibris.

### Основные директории

- `client/src/components/` - Переиспользуемые UI-компоненты
- `client/src/pages/` - Страницы приложения
- `client/src/hooks/` - Пользовательские React-хуки
- `client/src/lib/` - Вспомогательные библиотеки и утилиты
- `client/src/services/` - Сервисы для взаимодействия с API
- `client/src/types/` - Определения типов
- `client/src/assets/` - Статические ресурсы
- `client/src/styles/` - Стилевые файлы

### Стек технологий

- **React 19** - Библиотека для создания пользовательских интерфейсов
- **TypeScript** - Язык программирования с поддержкой статической типизации
- **Vite** - Инструмент сборки
- **Tailwind CSS** - Фреймворк для стилизации
- **React Query** - Управление состоянием и кэширование данных
- **React Hook Form** - Управление формами
- **Wouter** - Роутинг
