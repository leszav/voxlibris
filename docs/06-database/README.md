# База данных

В этом разделе описаны компоненты базы данных приложения VoxLibris.

## Содержание

- [Схема данных](schema.md) - Структура таблиц и связей
- [Миграции](migrations.md) - Управление изменениями схемы
- [Запросы](queries.md) - Примеры SQL-запросов и использования Drizzle ORM

## Технологии

- **PostgreSQL** - Реляционная база данных
- **Drizzle ORM** - Объектно-реляционное отображение
- **Drizzle Studio** - Визуальный инструмент для просмотра схемы

## Обзор

Схема базы данных приложения спроектирована для поддержки функциональности социального чтения, включая управление пользователями, клубами, книгами и сессиями чтения.
