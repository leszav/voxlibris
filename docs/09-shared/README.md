# Общие компоненты

В этом разделе описаны общие компоненты и утилиты, используемые как в клиентской, так и в серверной части приложения VoxLibris.

## Содержание

- [Утилиты](utilities.md) - Общие утилиты и функции

## Обзор

Папка `shared/` содержит типы, схемы и утилиты, которые используются как на клиенте, так и на сервере. Это позволяет избежать дублирования кода и обеспечивает согласованность типов во всем приложении.
