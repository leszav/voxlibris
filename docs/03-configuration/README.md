# Конфигурация

В этом разделе описаны конфигурационные файлы и настройки приложения VoxLibris.

## Содержание

- [TypeScript](typescript.md) - Конфигурация TypeScript для фронтенда и бэкенда
- [ESLint](eslint.md) - Настройки линтера и правила кода
- [Vite](vite.md) - Конфигурация сборки Vite
- [Tailwind](tailwind.md) - Настройки Tailwind CSS
- [Переменные окружения](environment.md) - Управление переменными окружения

## Обзор конфигурации

Приложение VoxLibris использует современные инструменты сборки и разработки, которые требуют соответствующей настройки для корректной работы. В этом разделе представлены все необходимые конфигурационные файлы и объяснение их параметров.

## Общие принципы конфигурации

- Использование единого стиля кода во всем проекте
- Типизация всех компонентов приложения
- Управление конфигурацией через переменные окружения
- Автоматическая проверка качества кода
- Оптимизация сборки для различных сред
