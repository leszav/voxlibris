# Email-шаблоны

В этом разделе описаны шаблоны электронных писем, используемые приложением VoxLibris.

## Содержание

- [Список шаблонов](templates-list.md) - Доступные шаблоны и их назначение
- [Кастомизация](customization.md) - Настройка и изменение шаблонов

## Обзор

Приложение VoxLibris использует электронную почту для уведомления пользователей о важных событиях, таких как приглашения в клубы, сброс паролей, уведомления о сессиях чтения и другие события.

## Технологии

- **Nodemailer** - Библиотека для отправки электронной почты
- **React Email** - Для создания HTML-шаблонов писем с использованием React
- **Resend** - Сервис для отправки транзакционных писем (опционально)
