# ADR-013: Архитектура системы аудио и записей

## Название

Использование архитектуры с поддержкой различных форматов и стримингом для системы аудио и записей

## Статус

Принято

## Контекст

В платформе VoxLibris необходимо реализовать систему аудио и записей, обеспечивающую запись, обработку и трансляцию аудио во время сессий чтения. Система должна поддерживать:

- Поддержку различных аудиоформатов (MP3, M4A, FLAC, OGG)
- Трансляцию аудио в реальном времени
- Запись и хранение сессий чтения
- Синхронизацию аудио с текстом книги
- Интеграцию с системой сессий чтения
- Хранение в S3-совместимом хранилище
- Обработку и конвертацию форматов

Были рассмотрены следующие альтернативы:
- Поддержка только одного формата
- Поддержка нескольких форматов с конвертацией
- Использование внешнего сервиса для обработки аудио
- Прямая трансляция без записи

## Решение

Мы выбираем архитектуру с поддержкой нескольких форматов и возможностью стриминга аудио по следующим причинам:

1. **Совместимость**: Поддержка нескольких форматов обеспечивает совместимость с разными устройствами и браузерами.

2. **Качество**: Возможность выбора формата в зависимости от требований к качеству и пропускной способности.

3. **Производительность**: Стриминг позволяет обрабатывать большие файлы без загрузки целиком.

4. **Функциональность**: Возможность записи и повторного прослушивания сессий важна для пользователей.

5. **Интеграция**: Система интегрируется с сессиями чтения и синхронизацией с текстом.

## Последствия

### Положительные эффекты

- Поддержка различных сценариев использования (живое чтение, записи, профессиональные аудиокниги)
- Совместимость с разными устройствами и браузерами
- Возможность оптимизации качества и производительности
- Поддержка синхронизации с текстом книги

### Негативные эффекты

- Усложнение архитектуры из-за поддержки нескольких форматов
- Необходимость обработки и конвертации аудио
- Потребность в большем объеме хранилища для записей
- Повышенные требования к пропускной способности

### Отказ от других альтернатив

- **Один формат**: Отклонен из-за ограничений совместимости
- **Внешние сервисы**: Отклонены из-за требований к контролю и интеграции
- **Только прямая трансляция**: Отклонена из-за необходимости записей для повторного прослушивания
