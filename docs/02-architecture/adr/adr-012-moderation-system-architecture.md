# ADR-012: Архитектура системы модерации и безопасности

## Название

Использование комбинированного подхода с автоматической и ручной модерацией для системы модерации и безопасности

## Статус

Принято

## Контекст

В платформе VoxLibris необходимо реализовать систему модерации и безопасности, обеспечивающую контроль качества контента, соблюдение правил платформы и защиту пользователей. Система должна поддерживать:

- Автоматическую фильтрацию нежелательного контента
- Ручную модерацию по жалобам пользователей
- Систему рейтингов для пользователей и контента
- Интеграцию с системой аутентификации
- Журналирование модерационных действий
- Защиту от атак и злоупотреблений

Были рассмотрены следующие альтернативы:
- Только автоматическая модерация
- Только ручная модерация
- Комбинированный подход с автоматической фильтрацией и ручной проверкой
- Использование внешних сервисов модерации

## Решение

Мы выбираем комбинированный подход с автоматической фильтрацией и ручной модерацией по следующим причинам:

1. **Эффективность**: Автоматическая фильтрация обрабатывает основной объем нежелательного контента.

2. **Точность**: Ручная модерация используется для сложных случаев, требующих человеческого суждения.

3. **Масштабируемость**: Комбинированный подход позволяет масштабировать систему с ростом платформы.

4. **Безопасность**: Многоуровневый подход обеспечивает лучшую защиту от различных угроз.

5. **UX**: Баланс между безопасностью и удобством использования для добросовестных пользователей.

## Последствия

### Положительные эффекты

- Высокий уровень безопасности платформы
- Эффективная фильтрация основного объема нежелательного контента
- Возможность обработки сложных случаев
- Поддержка доверия пользователей к платформе

### Негативные эффекты

- Усложнение архитектуры системы
- Необходимость управления модераторами
- Потребность в системе управления жалобами
- Возможные задержки при ручной модерации

### Отказ от других альтернатив

- **Только автоматическая модерация**: Отклонена из-за высокого уровня ложных срабатываний
- **Только ручная модерация**: Отклонена из-за масштабируемости
- **Внешние сервисы**: Отклонены из-за специфики платформы и конфиденциальности
