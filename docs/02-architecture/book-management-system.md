# Система хранения и управления книгами

## Обзор

Модуль системы хранения и управления книгами - это центральная часть приложения VoxLibris, отвечающая за загрузку, хранение, обработку и предоставление книг пользователям. Система поддерживает форматы EPUB и FB2, а также обеспечивает безопасное хранение и доступ к книгам.

## Компоненты системы

### 1. Personal Books Routes ([server/personal-books-routes.ts](file:///home/odmen/DEV/voxlibris/server/personal-books-routes.ts))

Маршруты для управления личными книгами пользователя:

- `/api/v1/user/books` - получение книг пользователя
- `/api/v1/user/books/upload` - загрузка новой книги
- `/api/v1/user/books/:id` - получение информации о книге
- `/api/v1/user/books/:id/delete` - удаление книги
- `/api/v1/user/books/:id/share` - настройка доступа к книге

### 2. Club Books Routes ([server/club-books-routes.ts](file:///home/odmen/DEV/voxlibris/server/club-books-routes.ts))

Маршруты для управления книгами клуба:

- `/api/v1/clubs/:clubId/books` - получение книг клуба
- `/api/v1/clubs/:clubId/books/add` - добавление книги в клуб
- `/api/v1/clubs/:clubId/books/:bookId/remove` - удаление книги из клуба
- `/api/v1/clubs/:clubId/books/:bookId/current` - установка книги как текущей для чтения

### 3. Book Parser ([server/book-parser.ts](file:///home/odmen/DEV/voxlibris/server/book-parser.ts))

Сервис для парсинга и обработки книг:

- Извлечение метаданных из файлов
- Разбор структуры книги (главы, содержание)
- Генерация обложки (если отсутствует)
- Конвертация форматов

### 4. File Storage ([server/file-storage.ts](file:///home/odmen/DEV/voxlibris/server/file-storage.ts))

Служба для безопасного хранения файлов:

- Загрузка файлов в S3-совместимое хранилище
- Удаление файлов из хранилища
- Получение URL-адресов для скачивания
- Проверка безопасности файлов

## Архитектура системы книг

### Поддерживаемые форматы

Система поддерживает следующие форматы книг:

1. **EPUB (.epub)**:
   - Полная поддержка метаданных
   - Поддержка обложек
   - Разбор глав и структуры

2. **FB2 (.fb2)**:
   - Поддержка основных метаданных
   - Поддержка обложек
   - Разбор глав

### Безопасность загрузки

Система имеет несколько уровней проверки:

1. **Проверка расширения файла**: только .epub и .fb2
2. **Проверка MIME-типа**: соответствие реальному формату файла
3. **Проверка размера файла**: максимальный размер 50MB
4. **Проверка magic numbers**: проверка настоящего формата файла
5. **Проверка на вирусы**: через внешние сервисы (в будущем)

## Основные функции

### Загрузка книги

1. Проверка аутентификации пользователя
2. Валидация файла (формат, размер, безопасность)
3. Сохранение файла во временное хранилище
4. Парсинг книги (извлечение метаданных, обложки)
5. Создание записи книги в базе данных
6. Сохранение файла в постоянное хранилище
7. Обновление информации в базе данных

### Обработка книги

При загрузке система выполняет:

1. **Извлечение метаданных**:
   - Название книги
   - Автор(ы)
   - ISBN
   - Описание
   - Жанр
   - Язык

2. **Обработка обложки**:
   - Извлечение из файла
   - Конвертация в подходящий формат
   - Оптимизация размера
   - Загрузка в хранилище

3. **Разбор структуры**:
   - Выделение глав
   - Создание оглавления
   - Разбиение на фрагменты для стриминга

### Управление доступом

Система поддерживает различные уровни доступа к книгам:

1. **Личные книги**:
   - Доступ только владельцу
   - Возможность предоставления доступа другим пользователям

2. **Клубные книги**:
   - Доступ только участникам клуба
   - Возможность предоставления доступа другим клубам

3. **Публичные книги**:
   - Доступ всем пользователям
   - Только для чтения

### Поиск книг

Система поддерживает поиск по:

- Названию книги
- Автору
- ISBN
- Описанию
- Жанрам

## Интеграция с другими модулями

### Система аутентификации

- Проверка прав доступа к книгам
- Ограничение доступа к личным книгам
- Проверка участия в клубах для доступа к клубным книгам

### Система клубов

- Доступ к книгам клуба только для участников
- Управление текущей книгой чтения
- Совместное чтение книг

### Система чтения

- Предоставление содержимого глав для чтения
- Обновление прогресса чтения
- Синхронизация между устройствами

### Система хранения

- Загрузка и хранение файлов книг
- Хранение обложек
- Генерация URL-адресов для доступа

## Безопасность

- Проверка безопасности файлов при загрузке
- Защита от Path Traversal атак
- Ограничение доступа к книгам на основе ролей
- Валидация MIME-типов
- Ограничение размера файлов

## Возможные проблемы и решения

### Проблемы с форматами

- **Неподдерживаемый формат**: проверка перед загрузкой
- **Поврежденный файл**: валидация структуры файла
- **Некорректные метаданные**: обработка исключений при парсинге

### Проблемы с хранением

- **Ограничение по размеру**: ограничение на 50MB
- **Доступ к файлам**: проверка прав перед предоставлением URL
- **Удаление файлов**: каскадное удаление файлов и записей в БД

### Проблемы с производительностью

- **Большие книги**: постраничная загрузка контента
- **Множество пользователей**: кэширование метаданных
- **Загрузка обложек**: оптимизация размера и формата

## Тестирование

Модуль управления книгами должен быть протестирован на:

- Корректность загрузки и парсинга различных форматов
- Безопасность загрузки файлов
- Права доступа к книгам
- Работу поиска книг
- Интеграцию с другими частями приложения
- Обработку граничных случаев (ограничения по размеру, дубликаты и т.д.)
