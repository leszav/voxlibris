# Серверная часть

В этом разделе описаны компоненты серверной части приложения VoxLibris.

## Содержание

- [Маршруты](routes.md) - Описание API-маршрутов и их функциональности
- [Сервисы](services.md) - Бизнес-логика и сервисные слои приложения
- [Middleware](middleware.md) - Промежуточное программное обеспечение
- [Контроллеры](controllers.md) - Обработчики запросов
- [API Reference](api-reference.md) - Справочник по API

## Архитектура серверной части

Серверная часть приложения реализована с использованием Express.js и TypeScript. Она обеспечивает API для клиентской части и обрабатывает всю бизнес-логику приложения.

### Основные директории

- `server/routes/` - API-маршруты
- `server/services/` - Сервисные слои
- `server/middleware/` - Промежуточные обработчики
- `server/repositories/` - Работа с базой данных
- `server/config/` - Конфигурационные файлы
- `server/utils/` - Вспомогательные утилиты

### Стек технологий

- **Express.js** - Веб-фреймворк
- **TypeScript** - Язык программирования
- **Drizzle ORM** - Объектно-реляционное отображение
- **Socket.IO** - Реальное время
- **JWT** - Аутентификация
- **PostgreSQL** - База данных
