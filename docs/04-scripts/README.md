В этом разделе описаны скрипты сборки, разработки и деплоя приложения VoxLibris.

## Содержание

- [Разработка](development.md) - Скрипты для локальной разработки
- [Тестирование](testing.md) - Скрипты для запуска тестов
- [Деплой](deployment.md) - Скрипты для деплоя приложения
- [Миграции](migrations.md) - Скрипты для управления миграциями базы данных

## Обзор скриптов

Приложение VoxLibris использует pnpm для управления зависимостями и запуска скриптов. Все скрипты определены в `package.json` в корне проекта.

## Основные скрипты

- `dev`: Запуск приложения в режиме разработки
- `build`: Сборка приложения для продакшена
- `start`: Запуск собранного приложения
- `lint`: Проверка кода с помощью ESLint
- `check`: Проверка типов с помощью TypeScript
- `test`: Запуск тестов
- `db:push`: Применение миграций к базе данных
- `init-storage`: Инициализация файлового хранилища
