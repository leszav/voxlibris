# Технологический стек

## Обзор технологий

VoxLibris использует современный стек технологий, обеспечивающий высокую производительность, масштабируемость и удобство разработки.

## Frontend

- **React 19**: Библиотека для создания пользовательских интерфейсов
- **TypeScript**: Язык программирования, надстройка над JavaScript с поддержкой статической типизации
- **Vite**: Быстрый сборщик модулей для современных веб-проектов
- **Tailwind CSS**: CSS-фреймворк с утилитами для быстрой стилизации
- **shadcn/ui**: Коллекция компонентов для создания пользовательского интерфейса
- **React Query**: Библиотека для управления состоянием и кэшированием данных
- **react-hook-form**: Библиотека для работы с формами

## Backend

- **Node.js 20+**: Среда выполнения JavaScript
- **Express.js**: Фреймворк для создания веб-приложений на Node.js
- **TypeScript**: Язык программирования с поддержкой статической типизации
- **Socket.IO**: Библиотека для реального времени через WebSocket
- **pino**: Быстрый и простой логгер для Node.js

## База данных

- **PostgreSQL 14+**: Мощная реляционная СУБД с открытым исходным кодом
- **Drizzle ORM**: Объектно-реляционное отображение с поддержкой TypeScript
- **Drizzle Kit**: Инструментарий для миграций и схемы базы данных

## Файловое хранилище

- **AWS S3-совместимое хранилище**: Для хранения книг, обложек и аудиозаписей
- **MinIO**: Высокопроизводительное хранилище объектов, совместимое с S3 (локальная разработка)

## Безопасность

- **Helmet**: Защита заголовков HTTP
- **express-rate-limit**: Ограничение количества запросов к API
- **bcrypt**: Хеширование паролей
- **jsonwebtoken**: Работа с JWT-токенами
- **dompurify**: Очистка HTML для предотвращения XSS-атак

## Форматы и обработка данных

- **Zod**: Библиотека для валидации данных
- **epub2**: Библиотека для обработки EPUB-файлов
- **sharp**: Библиотека для обработки изображений
- **multer**: Обработка multipart/form-data для загрузки файлов
- **file-type**: Определение типа файла по содержимому

## Сборка и разработка

- **pnpm 9**: Быстрый и эффективный менеджер пакетов
- **esbuild**: Быстрый сборщик JavaScript
- **globals**: Глобальные переменные для JavaScript
- **@types/node**: Типы для Node.js
- **@typescript-eslint/parser**: Парсер для ESLint
- **@vitejs/plugin-react**: Плагин для поддержки React в Vite

## Тестирование

- **Node.js built-in test runner**: Встроенный тестовый раннер Node.js
- **@types/...**: Типы для различных библиотек

## Дополнительные зависимости

- **cors**: Поддержка CORS для безопасного доступа к API
- **dotenv**: Управление переменными окружения
- **pg**: PostgreSQL клиент для Node.js
- **react-day-picker**: Компонент для выбора дат
- **recharts**: Библиотека для создания диаграмм
- **redis**: Клиент для Redis (для rate limiting)
- **rate-limit-redis**: Хранилище для rate limiting в Redis
- **wouter**: Роутер для React приложений
