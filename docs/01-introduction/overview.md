# Обзор проекта

## Назначение

VoxLibris — это платформа для социального чтения, предназначенная для создания и участия в книжных клубах, где книги читаются вслух профессиональными чтецами или другими пользователями. Платформа обеспечивает интерактивное чтение с возможностью общения, задавания вопросов и получения реакций от других участников сессии.

## Цели проекта

- Обеспечить качественный опыт совместного чтения книг
- Создать платформу для образования и культурного обмена
- Обеспечить доступ к литературе для людей с нарушениями зрения
- Поддерживать сообщество читателей и литературоведов
- Предоставить инструменты для организации образовательных мероприятий

## Функциональные возможности

### Чтение и аудио

- Поддержка различных форматов книг (EPUB, FB2)
- Возможность чтения текста с синхронизацией прогресса
- Поддержка аудио-чтения с возможностью записи сессий
- Воспроизведение записанных сессий
- Синхронизация аудио с текстом книги

### Книжные клубы

- Создание и управление книжными клубами
- Приглашения участников
- Управление ролями (владелец, модератор, участник)
- Совместное чтение книг в клубе
- Обсуждения книг и глав

### Коммуникация

- Чат во время сессий чтения
- Система реакций для быстрой обратной связи
- Система вопросов для более формального взаимодействия
- Уведомления о событиях и изменениях

### Пользовательские функции

- Персональные библиотеки
- Закладки и заметки
- История чтения
- Настройка профиля и предпочтений

## Архитектурные особенности

- Многоуровневая архитектура с четким разделением ответственности
- Использование современных технологий (React 19, TypeScript, Node.js)
- Безопасная аутентификация с JWT-токенами
- Поддержка реального времени через WebSocket
- Масштабируемая система хранения файлов

## Технологии

- **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS
- **Backend**: Express.js + TypeScript + Node.js 20+
- **Database**: PostgreSQL 14+ + Drizzle ORM
- **Real-time**: WebSocket (Socket.IO) для синхронизации чтения
- **Authentication**: JWT токены с refresh-токенами
- **File Storage**: AWS S3-совместимое хранилище (MinIO)
