# Пользовательское руководство

## Введение

VoxLibris - это платформа для социального чтения, позволяющая пользователям создавать и участвовать в книжных клубах, где книги читаются вслух профессиональными чтецами или другими пользователями. Это руководство поможет вам освоить основные функции платформы.

## Регистрация и аутентификация

### Создание аккаунта

1. Откройте сайт VoxLibris в вашем браузере
2. Нажмите кнопку "Регистрация"
3. Введите ваш email и придумайте пароль
4. Подтвердите ваш email, перейдя по ссылке из письма
5. Ваш аккаунт готов к использованию

### Вход в систему

1. Перейдите на страницу входа
2. Введите ваш email и пароль
3. Нажмите "Войти"
4. После входа вы попадете на главную страницу

### Восстановление пароля

1. Нажмите "Забыли пароль?" на странице входа
2. Введите ваш email
3. Перейдите по ссылке из письма
4. Введите новый пароль

## Основные функции

### Поиск и выбор книг

1. Перейдите на страницу каталога книг
2. Используйте поиск для нахождения интересующей вас книги
3. Просмотрите описание, обложку и отзывы
4. Выберите книгу для чтения или добавьте в избранное

### Чтение книг

1. Выберите книгу из каталога
2. Начните чтение в режиме "один":
   - Прокручивайте текст в своем темпе
   - Делайте заметки и закладки
   - Отслеживайте свой прогресс

### Присоединение к клубам

1. Перейдите в раздел "Клубы"
2. Найдите интересующий вас клуб
3. Нажмите "Присоединиться"
4. После подтверждения вы сможете участвовать в сессиях

### Создание собственного клуба

1. Нажмите "Создать клуб" в меню
2. Заполните информацию о клубе:
   - Название и описание
   - Тип (открытый/закрытый)
   - Максимальное количество участников
3. Пригласите друзей с помощью ссылок-приглашений
4. Добавьте книги для чтения

## Совместное чтение

### Участие в сессиях чтения

1. Перейдите в ваш клуб
2. Проверьте расписание сессий
3. Присоединитесь к сессии за 5-10 минут до начала
4. Во время сессии:
   - Следуйте за текстом, который читает ведущий
   - Используйте чат для общения с другими участниками
   - Отправляйте реакции на происходящее
   - Задавайте вопросы через систему вопросов

### Создание сессии чтения

1. Если вы модератор или владелец клуба:
2. Перейдите в раздел управления клубом
3. Нажмите "Создать сессию"
4. Выберите книгу и время
5. Пригласите участников
6. Начните сессию в назначенное время

### Запись сессий

1. В процессе сессии чтения ведущий может включить запись
2. Запись сохраняется в библиотеке клуба
3. Участники могут прослушать запись позже
4. Записи доступны только участникам клуба

## Взаимодействие с другими пользователями

### Общение в чатах

1. Используйте чат во время сессии чтения
2. Общайтесь в клубном чате
3. Уважайте других участников
4. Следуйте правилам общения в клубе

### Отправка реакций

1. Во время чтения используйте кнопки реакций
2. Реакции помогают ведущему понять вашу реакцию
3. Реакции видят все участники сессии
4. Используйте реакции уместно

### Задавание вопросов

1. Используйте систему вопросов для задания вопросов
2. Вопросы поступают ведущему или модератору
3. Вопросы могут быть анонимными
4. Другие участники могут голосовать за важные вопросы

## Управление профилем

### Редактирование профиля

1. Перейдите на страницу вашего профиля
2. Нажмите "Редактировать"
3. Обновите информацию:
   - Имя пользователя
   - Аватар
   - Описание
   - Предпочтения
4. Сохраните изменения

### Настройка уведомлений

1. Перейдите в настройки уведомлений
2. Выберите, какие уведомления вы хотите получать:
   - Email-уведомления
   - Push-уведомления
   - Внутренние уведомления
3. Настройте частоту и время получения

### Настройка приватности

1. Перейдите в настройки приватности
2. Управляйте видимостью вашей информации
3. Настройте доступ к вашей активности
4. Управляйте возможностью получения приглашений

## Загрузка и добавление книг

### Загрузка собственных книг

1. Перейдите в раздел "Мои книги"
2. Нажмите "Загрузить книгу"
3. Выберите файл в формате EPUB или FB2
4. Заполните метаданные книги:
   - Название
   - Автор
   - Описание
   - Обложка
5. Книга будет доступна вам и вашим клубам

### Добавление книг в клуб

1. Перейдите в ваш клуб
2. Откройте раздел "Библиотека"
3. Нажмите "Добавить книгу"
4. Выберите из ваших книг или каталога
5. Книга станет доступна участникам клуба

## Планирование и расписание

### Просмотр расписания

1. Перейдите на страницу клуба
2. Откройте раздел "Расписание"
3. Просмотрите предстоящие сессии
4. Установите напоминания о сессиях

### Планирование сессий

1. Если вы модератор клуба:
2. Используйте календарь для планирования
3. Назначьте дату и время сессии
4. Выберите книгу для чтения
5. Пригласите участников
6. Система отправит уведомления участникам

## Поддержка и помощь

### Поиск помощи

1. Посетите раздел "Помощь" на сайте
2. Ознакомьтесь с часто задаваемыми вопросами
3. Используйте поиск по справке
4. Обратитесь в службу поддержки

### Сообщение о проблемах

1. Если вы столкнулись с технической проблемой:
2. Перейдите в раздел "Сообщить о проблеме"
3. Опишите проблему подробно
4. Укажите шаги для воспроизведения
5. Приложите скриншоты, если возможно

## Безопасность и приватность

### Защита аккаунта

1. Используйте сложный пароль
2. Не делитесь своими учетными данными
3. Выходите из аккаунта на общих компьютерах
4. Регулярно проверяйте активные сессии

### Правила поведения

1. Уважайте других пользователей
2. Не публикуйте неприемлемый контент
3. Следуйте правилам конкретного клуба
4. Сообщайте о нарушениях модераторам
