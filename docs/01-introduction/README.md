## Что такое VoxLibris?

VoxLibris — это платформа для социального чтения, позволяющая пользователям создавать и участвовать в книжных клубах, где книги читаются вслух профессиональными чтецами или другими пользователями. Платформа обеспечивает интерактивное чтение с возможностью общения, задавания вопросов и получения реакций от других участников сессии.

## Основные возможности

- Создание и участие в книжных клубах
- Совместное чтение книг с синхронизацией прогресса
- Поддержка аудио-чтения с возможностью записи сессий
- Поддержка различных форматов книг (EPUB, FB2)
- Система коммуникации во время чтения (чат, реакции, вопросы)
- Индивидуальные и групповые сессии чтения
- Система уведомлений и планирования сессий
