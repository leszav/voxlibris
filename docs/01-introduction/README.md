# Введение в VoxLibris

Этот раздел содержит основную информацию о проекте VoxLibris для новых разработчиков и участников.

## О проекте

VoxLibris — это инновационная платформа для социального чтения, которая объединяет людей для совместного чтения и обсуждения книг.

### Ключевые возможности

- 📚 **Социальное чтение** - читайте книги вместе с другими пользователями
- 🏠 **Книжные клубы** - создавайте и управляйте сообществами
- 🎙️ **Голосовое вещание** - читайте вслух для других участников
- 💬 **Обсуждения** - обменивайтесь мнениями о прочитанном
- 📊 **Прогресс и достижения** - отслеживайте свою активность

## Технологический стек

### Frontend
- React 18 с TypeScript
- Vite для сборки
- Tailwind CSS для стилей
- Zustand для управления состоянием

### Backend
- Node.js с TypeScript
- Express.js для API
- Drizzle ORM
- PostgreSQL
- Redis для кэширования
- WebSockets для real-time

### Инфраструктура
- Docker для контейнеризации
- MinIO для файлового хранения
- Nginx для reverse proxy
