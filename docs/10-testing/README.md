# Тестирование

В этом разделе описаны подходы и практики тестирования приложения VoxLibris.

## Содержание

- [Unit-тесты](unit-tests.md) - Тестирование отдельных функций и компонентов
- [Интеграционные тесты](integration-tests.md) - Тестирование взаимодействия между компонентами
- [E2E-тесты](e2e-tests.md) - Тестирование сквозных сценариев

## Обзор

Приложение VoxLibris использует многоуровневый подход к тестированию, включающий unit-тесты, интеграционные тесты и E2E-тесты. Это позволяет обеспечить высокое качество кода и надежность приложения.
